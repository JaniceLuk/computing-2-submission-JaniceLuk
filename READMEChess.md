# Chess

**CID**: [02604065]

This project is a web-based implementation of Chess, developed using HTML, CSS and JavaScript as part of the Computing 2 Applications coursework.

## Features

The application supports the standard rules of chess, including:

* Legal movement for all chess pieces
* Piece captures
* Check detection
* Checkmate detection
* Stalemate detection
* Automatic pawn promotion to Queen
* Display of captured pieces
* Game reset
* Responsive layout for smaller screens

## Project Structure

The project consists of the following files:

* `chess.js` – Implements the chess game logic and rules.
* `main.js` – Handles rendering and user interaction.
* `chess.css` – Styles the web application.
* `index.html` – Defines the page structure.
* `tests/` – Contains Mocha unit tests for the game logic.
